Nesta página é possível verificar os novos casos de Covid-19 na região cadastrada para o sistema.
É possível visualizar os 7 últimos novos casos registrados e a previsão para mais 7 dias a partir do caso mais recente.

Definição:
Em lilás é possível ver a quantidade de casos novos nas respectivas datas.
Em vermelho é possível ver a quantidade de casos novos previstos pelo sistema ConEg.
